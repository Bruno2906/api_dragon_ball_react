# 🐉 Dragon Ball Universe

Projeto desenvolvido em React consumindo a API pública de Dragon Ball para exibir personagens do universo Dragon Ball Z.

##  Funcionalidades

-  Busca de personagens em tempo real
- ⭐ Sistema de favoritos
- 📄 Página de detalhes do personagem
- ⚡ Skeleton loading
- 📱 Layout responsivo
- 🌐 Deploy online com Vercel
- 🎨 Interface estilizada com tema Dragon Ball

---

## 🛠️ Tecnologias utilizadas

- React
- Vite
- React Router DOM
- Context API
- JavaScript
- CSS3
- API REST
- Vercel
